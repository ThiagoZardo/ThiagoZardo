## Sobre mim

Sou desenvolvedor full stack com experiência em frontend, backend, mobile, APIs, DevOps e cloud. Trabalho com tecnologias como React, React Native, Node.js, TypeScript, NestJS, PHP, .NET, C#, bancos de dados relacionais e não relacionais, além de ferramentas de infraestrutura e automação.
